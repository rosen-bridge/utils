## Introduction

`@rosen-bridge/ergo-box-selection` Typescript package to select enough Ergo boxes for required amount of assets

## Installation

npm:

```sh
npm i @rosen-bridge/ergo-box-selection
```

yarn:

```sh
yarn add @rosen-bridge/ergo-box-selection
```

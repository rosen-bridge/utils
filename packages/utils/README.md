## Introduction

`@rosen-bridge/utils` contains some common utility functions and apis that are used
inside other Rosen bridge projects.

## Installation

npm:

```sh
npm i @rosen-bridge/utils
```

yarn:

```sh
yarn add @rosen-bridge/utils
```

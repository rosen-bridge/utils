## Table of contents

## Introduction

`@rosen-bridge/abstract-logger` contains abstract logger class, the base type
for loggers used in all Rosen packages.

## Installation

npm:

```sh
npm i @rosen-bridge/abstract-logger
```

yarn:

```sh
yarn add @rosen-bridge/abstract-logger
```

## Introduction

`@rosen-bridge/minimum-fee` Typescript package to build and get minimum fee of the bridge for supported tokens from blockchain

## Installation

npm:

```sh
npm i @rosen-bridge/minimum-fee
```

yarn:

```sh
yarn add @rosen-bridge/minimum-fee
```
